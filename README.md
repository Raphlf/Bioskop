# bioskop-app

Simple cinema ticket booking web app (vanilla HTML/CSS/JS + PHP + MySQL)

Instructions:
1. Import `sql/schema.sql` into your MySQL server.
2. Adjust `src/config.php` DB credentials and `BASE_URL`.
3. Set your web server document root to `public/` or serve that folder.
4. Make sure `public/assets/uploads` is writable.
5. Register an admin user via `register.php` and change role to 'admin' in DB, or insert admin manually.
